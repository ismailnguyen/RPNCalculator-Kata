# RPNCalculator-Kata

``RPNCalculator``

public double calculate(String formula) où la formule est une expression valide en notation polonaise inversée.

```
5 3 + => 5+3 => 8

6 2 / => 6/2 => 3

5 2 - 7 + => 10

7 5 2 - + => 10

3 5 8 x 7 + x => 3x((5x8)+7) => 141

3 4 2 1 + x + 2 / => (3 + (4 x (2+1)))/2 => 7.5

1 2 + 4 × 5 + 3 − => ((1+2) x 4) + 5 - 3 => 14
```
